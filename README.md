# Portafolio

## Descripción del proyecto

Este es mi sitio web de portafolio personal, desarrollado como proyecto del
Sprint 4 del bootcamp de Desarrollo Web de TripleTen. El sitio presenta mi
perfil profesional, mis habilidades técnicas y los proyectos que he
desarrollado a lo largo del bootcamp, junto con mis enlaces de contacto y mi
currículum descargable.

El diseño fue implementado siguiendo un brief de Figma, con un enfoque
completo en diseño responsivo para adaptarse a resoluciones de escritorio,
tablet y móvil.

## Funcionalidad

- Sección de perfil con foto, biografía y enlaces (CV descargable y GitHub).
- Sección de habilidades y herramientas con iconografía.
- Sección de proyectos destacados con tarjetas que incluyen imagen, título y
  descripción.
- Sección de contacto con enlaces a redes sociales.
- Diseño completamente responsivo (desktop, tablet y móvil).

## Tecnologías y técnicas utilizadas

- HTML5 semántico
- CSS3 (Flexbox, Grid, `calc()`, degradados lineales y radiales)
- Diseño responsivo con media queries (puntos de ruptura en 767px y 1023px)
- Tipografías personalizadas (`Open Sans` y `Archivo Black`) conectadas
  localmente mediante `@font-face`
- Git y GitHub para control de versiones
- Despliegue mediante GitHub Pages
- Validación de código con el validador W3C
- Uso de herramientas de IA para la resolución de efectos visuales avanzados
  (degradados de fondo y superposición de capas)
